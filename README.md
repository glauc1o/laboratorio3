# laboratorio

Nesse repositório se encontra a resolução do laboratório 3, relacionado a implementação do Decoder. No arquivo "decoder.py" foram implementadas as funções: create_causal_mask; cross_attention; generate_next_token.

Em "attention.py", a função softmax, implementada no laboratório 1, é utilizada pelo decoder.

"decoder_test.py" é responsável pelos testes das funções citadas.

Bibliotecas usadas: numpy.

Ferramentas de pesquisa usadad: Gemini.s
